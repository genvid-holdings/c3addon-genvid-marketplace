## To develop

```bash
npx http-server src --cors
```

Because of the restricted CSP in Construct3, make sure to use http://localhost:8080/addon.json instead of 127.0.0.1.

## To build

```bash
npm run all:{platform}
```

where platform is either `windows` or `posix`.
